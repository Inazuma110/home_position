# HomePosition
応用プログラミングIIIの課題。
Brainf**kの亜種をつくる。HomePositionベースで打ちやすい(多分)


## 使い方
コンパイル後に、
```sh
java Main ファイル名
```
または、
```sh
java -jar home_position.jar ファイル名
```
で実行。
how_to.pdfにもう少し丁寧にまとめてあります。
